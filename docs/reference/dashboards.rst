Live Dashboards
===============

Everything under ``https://ubc-fresh.github.io/fhops/`` is rebuilt automatically on every
successful ``main`` run. These artefacts mirror the telemetry tuning workflows described in
:doc:`../howto/telemetry_tuning` so refer to that guide for deeper context.

Direct links:

- **Global telemetry history** – interactive trend chart.
  https://ubc-fresh.github.io/fhops/telemetry/history_summary.html
- **Latest per-scenario history snapshot** – leaderboard for the latest report label.
  https://ubc-fresh.github.io/fhops/telemetry/latest_history_summary.md
- **Latest tuner report** – raw Markdown table for code reviews.
  https://ubc-fresh.github.io/fhops/telemetry/latest_tuner_report.md
- **Comparison / leaderboard tables** – win counts and runtime ratios.
  https://ubc-fresh.github.io/fhops/telemetry/latest_tuner_comparison.md
  https://ubc-fresh.github.io/fhops/telemetry/latest_tuner_leaderboard.md
- **Per-bundle difficulty tables** – gap + success rates per tier.
  https://ubc-fresh.github.io/fhops/telemetry/tuner_difficulty.md

See :ref:`telemetry_dashboard_interpretation` for guidance on how to respond to each signal.
